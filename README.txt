# Beauté Des Ongles - Landing Page

This project represents the landing page for "Beauté Des Ongles", a nail beauty salon offering various nail care services.

## Features

- Fixed Header with Company Logo and Navigation links.
- Responsive Contact Form to get specific requests or questions.
- List of key features that describe the salon's unique services.
- Video section (currently using a placeholder).
- List of services offered with their respective prices.

## Technologies Used

- HTML5
- CSS3

## Upcoming Changes

1. Improving the CSS styles for better UI and UX.
2. Possible functionality addition using JavaScript.

## How to Use

Just open the HTML file in a browser to view the landing page. For editing, use any code editor to make changes to the HTML and CSS files.

## Fonctionnalités

- En-tête fixe avec le logo de la société et des liens de navigation.

- Liste des principales caractéristiques décrivant les services uniques du salon.
- Section vidéo (actuellement utilisant un espace réservé).
- Liste des services proposés avec leurs prix respectifs.

## Technologies Utilisées

- HTML5
- CSS3
